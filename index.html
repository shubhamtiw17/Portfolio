Exploring Hyperparameter Optimization for Enhancing the Privacy-Utility Trade-off for the Synthetization of Tabular Data
Erkundung der Hyperparameter-Optimierung zur Verbesserung des Datenschutz-Nutzen-Kompromisses bei der Synthetisierung tabellarischer Daten

Designed and built SAGE (Synthetic data Analysis via Grid search Evaluation), a modular evaluation framework that benchmarks three generative model families: CTGAN, TVAE, and TabDDPM across a full hyperparameter grid on three medical datasets (Pima Diabetes, Obesity, Stroke). Each configuration was evaluated across resemblance, utility (Train on Synthetic, Test on Real), and privacy metrics including membership inference and attribute inference attacks.

Entwicklung von SAGE (Synthetic data Analysis via Grid search Evaluation), einem modularen Evaluierungsframework, das drei generative Modellfamilien: CTGAN, TVAE und TabDDPM auf einem vollständigen Hyperparameter-Grid mit drei medizinischen Datensätzen benchmarkt. Jede Konfiguration wurde anhand von Ähnlichkeit, Nutzbarkeit (Train on Synthetic, Test on Real) und Datenschutzmetriken bewertet.

Key finding: TabDDPM delivered the most stable and privacy-preserving results across all datasets and configurations, while TVAE reached peak utility but consistently fell into low-privacy zones. Utility improved by 20-50% through systematic hyperparameter optimisation.

Kernbefund: TabDDPM lieferte die stabilsten und datenschutzfreundlichsten Ergebnisse über alle Datensätze und Konfigurationen. TVAE erreichte die höchste Nutzbarkeit, fiel jedoch konstant in Niedrig-Datenschutz-Bereiche. Die Nutzbarkeit wurde durch systematische Optimierung um 20–50 % verbessert.
