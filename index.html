Exploring Hyperparameter Optimization for Enhancing the Privacy-Utility Trade-off for the Synthetization of Tabular Data

Erkundung der Hyperparameter-Optimierung zur Verbesserung des Datenschutz-Nutzen-Kompromisses bei der Synthetisierung tabellarischer Daten

Designed and built SAGE (Synthetic data Analysis via Grid search Evaluation) — a modular framework benchmarking CTGAN, TVAE, and TabDDPM across a full hyperparameter grid on 3 medical datasets (Pima Diabetes, Obesity, Stroke). Each configuration was evaluated across resemblance, utility (TSTR), and privacy (membership & attribute inference) metrics, with quadrant-based visualisation of the privacy–utility trade-off.

Entwicklung von SAGE (Synthetic data Analysis via Grid search Evaluation) — ein modulares Framework zum Benchmarking von CTGAN, TVAE und TabDDPM auf einem vollständigen Hyperparameter-Grid mit 3 medizinischen Datensätzen. Jede Konfiguration wurde hinsichtlich Ähnlichkeit, Nutzbarkeit (TSTR) und Datenschutz bewertet.

Key finding: TabDDPM achieved the most stable and privacy-preserving results across all datasets, while TVAE reached peak utility but consistently fell into low-privacy zones — making it unsuitable for sensitive health data without careful tuning. Utility improved by 20–50% through systematic optimisation.

Kernbefund: TabDDPM erzielte die stabilsten und datenschutzfreundlichsten Ergebnisse. TVAE erreichte die höchste Nutzbarkeit, fiel jedoch konstant in Niedrig-Datenschutz-Bereiche. Die Nutzbarkeit wurde durch systematische Optimierung um 20–50 % verbessert.
